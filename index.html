Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

Since there are multiple types of web-attacks, there are multiple things you can do against them: 2-step verification, sanitazion of the data to make sure it's in the right format for the system, using trusted 3rd party apps, using CI/CD to ensure that the code being deployed has passed a "quality check", having personell dedicated to network security, and last is setting up a timer for logging out in sessions where the user has not been active for a spesific timeperiod. There are more things you can do, but these are the examples that was in the course.
